# Bling Truth

![Gameplay](https://img.itch.zone/aW1nLzIzOTMzNjY3LmdpZg==/315x250%23c/xr4YHf.gif)

Короткая история о девушке, которая отправилась в забытую деревню на реке Горхон в поисках своих родителей.
Каждый шаг приближает её к истине, которую, возможно, лучше было бы не знать.

======================

Игра, созданная за 3 дня в рамках геймджема.
